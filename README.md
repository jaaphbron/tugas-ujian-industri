## Cara Menggunakan

1. **Tambahkan Foto (Opsional):**
   - Tambahkan foto ayam geprek di folder `images/` dengan nama:
     - `ayam-geprek.jpg` (untuk hero section)
     - `ayam-geprek-2.jpg` (untuk section tentang)
     - `ayam-geprek-3.jpg` (untuk section pesan)
   - Jika foto tidak ada, website akan menggunakan foto placeholder otomatis

2. **Konfigurasi WhatsApp (Opsional):**
   - Buka file `script.js`
   - Ganti nomor WhatsApp di baris dengan variabel `whatsappNumber` (default: 6281234567890)

3. Buka file `index.html` di browser
4. Website siap untuk dipresentasikan!

## Struktur File

- `index.html` - Struktur utama website
- `styles.css` - Styling dan desain
- `script.js` - Interaktivitas dan animasi
- `images/` - Folder untuk foto ayam geprek
- `README.md` - Dokumentasi

## Teknologi

Form pesan akan mengirim data ke WhatsApp dengan format yang rapi. Pastikan untuk:
- Mengganti nomor WhatsApp di `script.js` dengan nomor Anda
- Menguji form sebelum presentasi
